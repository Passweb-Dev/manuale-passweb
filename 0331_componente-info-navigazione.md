# COMPONENTE INFO NAVIGAZIONE



Il Componente **"Info Navigazione"** consente, di base, di inserire in
una qualsiasi pagina del sito le cosiddette "briciole di pane" mediante
le quali poter individuare esattamente l'attuale posizione dell'utente
all'interno del sito.

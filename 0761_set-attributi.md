# SET ATTRIBUTI



All'interno di questa sezione è possibile definire quale set di
attributi dovrà essere associato automaticamente alle nuove categorie
merceologiche importate all'interno del sito.

![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\catalogo_set_attributi.bmp](./assets/media/image40.png){width="5.804861111111111in"
height="3.5256944444444445in"}

E' possibile selezionare una delle seguenti opzioni:

- **Set Attributi Default:** in queste condizioni verrà assegnato, alle
  nuove categorie merceologiche, in fase di sincronizzazione, il set di
  attributi impostato come set di default.

![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\catalogo_set_attributi_2.bmp](./assets/media/image41.png){width="5.584722222222222in"
height="3.441666666666667in"}

> **ATTENZIONE!** Nel caso in cui non sia stato definito nessun Set di
> Default alle nuove categorie merceologiche non verrà associato, in
> fase di sincronizzazione, nessun Set di Attributi.

- **Set Attributi Categoria Padre:** in queste condizioni in fase di
  sincronizzazione verrà automaticamente associato alla nuova categoria
  merceologica importata all'interno del sito, il set di attributi
  definito per la categoria padre.

> Occorre inoltre considerare che:

- Nel caso in cui la categoria padre non abbia nessun set associato,
  alla nuova categoria merceologia verrà associato il set di default

- Nel caso in cui la nuova categoria merceologica non abbia alcun padre
  (categoria di primo livello) gli verrà associato il set di default

- Nel caso in cui, nelle due condizioni sopra citate, non dovesse essere
  presente un set di attributi di default, alla nuova categoria
  merceologica non verrà associato nessun set di attributi

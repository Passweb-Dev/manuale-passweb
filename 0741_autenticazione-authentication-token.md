parametri di configurazione del portale saranno sostanzialmente analoghi
a quelli indicati nel capitolo precedente con l'unica differenza
rappresentata dal fatto che al posto della chiave API per effetture
l'autenticazione tra le due piattaforme dovremo ora utilizzare il token
di accesso.

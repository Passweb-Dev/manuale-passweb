# PANNELLO RICERCA DOCUMENTI



La ricerca dei file gestiti all'interno della sezione Documenti
dell'area riservata può avvenire in due modi differenti:

- selezionando una delle cartelle presenti nella parte sinistra della
  pagina

- utilizzando il pannello "Ricerca Documenti" presente invece nella
  parte destra della pagina

Nel primo caso, una volta selezionata la cartella desiderata, verranno
automaticamente visualizzati, all'interno della griglia documenti, tutti
i file in essa contenuti

![](./assets/media/image109.png)

Per quel che riguarda invece il pannello di **Ricerca Documenti**,
questo potrà comportarsi in maniera diversa a seconda del fatto di aver
attivato o meno la gestione delle Web Api Passcom.

In particolare nel caso in cui si dovesse decidere di **non attivare le
Web Api**, il pannello "Ricerca Documenti" avrà al suo interno solamente
il campo "Titolo" e, di fatto, consentirà di ricercare unicamente i
documenti presenti all'interno di Cartelle Standard e/o di Cartelle
Azienda diverse però dalle Cartelle Docuvision

Una volta **attivate le Web Api**, all'interno del pannello "Ricerca
Documenti" troveremo invece, oltre al campo "**Titolo**" anche i campi
"**Data**", "**Classe**" e "**Tipo**", e soprattutto, tale pannello
potrà essere utilizzato per ricercare non solo i documenti presenti
all'interno delle Cartelle Standard ma anche quelli presenti all'interno
delle Cartelle Docuvision

Per maggiori informazioni in merito all'utilizzo del pannello di
"Ricerca Documenti" nei due casi sopra indicati si vedano anche i
successivi capitoli di questo manuale

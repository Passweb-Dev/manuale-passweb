# WISHLIST



La sezione **"Wishlist"**, accessibile dalla voce di menu principale
"**Utenti**" consente di:

- Visualizzare tutte le Wishlist di tipo "Lista Regalo" create dagli
  utenti del sito oltre che gli ordini generati a partire da esse --
  **Gestione Liste Regalo**

- Indicare le priorità selezionabili dall'utente in fase di creazione di
  una Lista Regalo -- **Priorità**

- Indicare le tipologie di Wishlist gestibili sul sito -- **Tipologia**

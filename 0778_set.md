# SET



La pagina **"Set di Opzioni"** accessibile dalla voce di menu
***"Catalogo -- Opzioni Articoli -- Set"***, consente di visualizzare e
gestire tutti i Set (insiemi) di opzioni di personalizzazione articolo
attualmente codificati oltre che, ovviamente, di crearne di nuovi.

![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\set_opzioni_elenco.bmp](./assets/media/image138.png)

I pulsanti presenti nella barra principale degli strumenti consentono
rispettivamente di:

**Modifica**
(![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\pulsante_modifica.bmp](./assets/media/image43.png) ): consente di modificare il set di
opzioni attualmente selezionato in elenco sia a livello di
configurazione che di custom option in esso presenti

**Elimina**
(![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\pulsante_elimina.bmp](./assets/media/image114.png) ): consente di eliminare il set di
opzioni attualmente selezionato in elenco

**Copia**
(![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\pulsante_copia.bmp](./assets/media/image128.png) ): consente di effettuare una copia del
set di opzioni attualmente selezionato in elenco

**Nuovo**
(![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\pulsante_nuovo.bmp](./assets/media/image88.png) ): consente di definire un nuovo Set di
Opzioni.

Per maggiori informazioni relativamente alla procedura di creazione di
un nuovo set di Opzioni si veda quanto indicato nel successivo capitolo
di questo manuale

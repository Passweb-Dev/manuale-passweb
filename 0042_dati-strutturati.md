# DATI STRUTTURATI



All'interno di questa sezione è possibile abilitare e configurare
l'utilizzo dei cosiddetti **dati strutturati**, relativamente alla
"parte Ecommerce" del sito web decidendo anche se implementarne
l'utilizzo mediante Microdati o JSON LD

![\\\\doctohelp\\D\\ManPassweb_Ecommerce_Sviluppo\\Videate\\microdati_1.bmp](./assets/media/image119.png)

**ATTENZIONE! l'implementazione dei dati strutturati mediante JSON LD
richiede l'attivazione del relativo modulo su Passstore**

Per maggiori informazioni in merito ai dati strutturati (cosa sono,
perché sono utili, come si implementano all'interno del sito Passweb
ecc...) si rimanda a quanto indicato all'interno del relativo capitolo
("*Dati Strutturati*") di questo manuale.

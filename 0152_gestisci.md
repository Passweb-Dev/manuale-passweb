# GESTISCI



Il pulsante **Gestisci** (
![](./assets/media/image153.png) ): consente di accedere alla maschera di
definizione delle caratteristiche strutturali del layout selezionato.

Cliccando su questo pulsante sarà quindi possibile: inserire appositi
Meta Tag nella sezione \< head \> di ogni pagina cui lo specifico layout
verrà associato, includere all'interno del layout specifici file CSS o
Javascript; scrivere codice Javascript o CSS personalizzato ecc ...

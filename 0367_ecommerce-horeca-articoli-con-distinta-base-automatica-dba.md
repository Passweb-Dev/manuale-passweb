# ECOMMERCE HO.RE.CA. -- ARTICOLI CON DISTINTA BASE AUTOMATICA (DBA)



Come avviene già per i siti Ecommerce collegati a Mexal, anche nel caso
di siti collegati ad uno dei gestionali Ho.Re.Ca. il Componente
**"Campionario E-commerce"** consente di gestire correttamente articoli
con Distinta Base Automatica

L'utilizzo di tale componente ai fini della vendita di articoli con DBA
è esattamente lo stesso che si avrebbe nel caso di siti collegati a
Mexal.

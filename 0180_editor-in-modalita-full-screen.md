# EDITOR IN MODALITA' FULL SCREEN



L'icona "**Massimizza** ( )" posta nell' ultima riga della barra degli
strumenti consente di aprire l'editor in modalità full screen aumentando
notevolmente lo spazio disponibile per l'area di lavoro.

![](./assets/media/image278.png)

Una volta attivata la modalità full screen lo stesso pulsante
"Massimizza" consente di riportare l'editor alle sue dimensioni
originali.

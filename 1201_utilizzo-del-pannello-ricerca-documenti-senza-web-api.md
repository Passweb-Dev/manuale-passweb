# UTILIZZO DEL PANNELLO RICERCA DOCUMENTI SENZA WEB API



Come indicato nel precedente capitolo di questo manuale nel caso in cui
si dovesse decidere di non attivare le Web Api, il pannello "Ricerca
Documenti" avrà al suo interno solamente il campo "Titolo" e, di fatto,
consentirà di ricercare unicamente i documenti presenti all'interno di
Cartelle Standard e/o di Cartelle Azienda diverse però dalle Cartelle
Docuvision

![](./assets/media/image110.png)

Una volta indicato il testo da ricercare cliccando sul pulsante
"**Vai**" verranno poi visualizzati, immediatamente al di sotto del
campo di ricerca, tutti i risultati che soddisfano il filtro impostato

![](./assets/media/image111.png)

**ATTENZIONE! la ricerca per titolo del documento è una ricerca di tipo
"contiene"**

Ogni risultato mostra il percorso completo del file che soddisfa il
filtro. Cliccando quindi su uno dei risultati in elenco verrà aperta la
relativa cartella e verranno visualizzati, in griglia, tutti i documenti
in essa contenuti

![](./assets/media/image112.png)

Il pulsante "**Annulla**" consente invece di ripulire e azzerare il
filtro precedentemente impostato.

**ATTENZIONE!** **nel momento in cui si dovesse decidere di non attivare
le Web Api, l'unica possibilità di accedere ai documenti gestiti
direttamente all'interno del gestionale, sarà quella di selezionare la
relativa Cartella Docuvision**
